# React API Fetching Project

## 📌 Overview
This project is part of **Experiment 3** for AIML coursework.  
It demonstrates how to build a **React.js application** that fetches data from a public API and displays it in a clean, modern user interface.

---

## 🎯 Objectives
- Understand React component structure
- Learn API integration using `fetch()` or `axios`
- Use React hooks: `useState()` and `useEffect()`
- Display dynamic data in a card-based UI
- Handle loading and error states
- Implement search functionality (bonus)

---

## 🧩 Mandatory React Concepts
The project uses:
- Functional Components
- `useState()` for state management
- `useEffect()` for side effects (API calls)
- API call using `fetch()`
- `map()` for rendering multiple items
- Conditional Rendering for loading/error states

---

## 🎨 UI & Design Features
- Clean modern layout
- Card-based design
- Responsive for mobile and desktop
- Hover effects on cards and buttons
- Proper spacing and alignment
- Search bar to filter records (bonus)

---

## 🌐 API Used
We fetch data from:
- `https://jsonplaceholder.typicode.com/users`

To make the project more relevant, names and cities are replaced with **Indian names and cities**.
